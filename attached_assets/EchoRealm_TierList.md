## RARITY KEY

| Rarity     | In-game label | Color          | How to get                                                    |
|------------|---------------|----------------|---------------------------------------------------------------|
| Common     | Common        | Grey           | Shops, early chests, starting area drops                      |
| Uncommon   | Uncommon      | Green          | Shops, mid-game chests, side quest rewards                    |
| Rare       | Rare          | Blue           | Chests only — must find in the world                         |
| Epic       | Epic          | Purple         | Boss/dungeon chests, late side quest rewards                  |
| Legendary  | Legendary     | Orange         | End-game content, major quest completions                     |
| Mythic     | Mortus        | Animated gold  | Crafted via Tomes Blessing + Empty Book only                  |
| Chromatic  | Chromatic     | Animated color | Challenge Board Color tier only (ch_creed_emblem tier)        |
| Challenge  | (various)     | Tier color     | Challenge Board rewards — slot-independent, carry into runs   |
| NG+        | NG+           | Gold shimmer   | Drop from Memory Wraith boss in New Game+ runs only           |
